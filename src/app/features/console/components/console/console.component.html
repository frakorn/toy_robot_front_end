Welcome to Toy Robot Simulator Terminal!
COMMAND LIST:
- PLACE will put the toy robot on the table in position X,Y and facing NORTH, SOUTH, EAST or WEST.
- The origin (0,0) is the SOUTH WEST most corner.
- The first valid command to the robot is the PLACE command
- MOVE will move the toy robot one unit forward in the direction it is currently facing.
- LEFT and RIGHT will rotate the robot 90 degrees in the specified direction without changing the position of the robot.
- REPORT will show a modal with the X,Y and F of the robot
- CLOSE will close Robot Simulator window
- CLEAR will clear simulator consolle
- Type your command below:
toy-robot@terminal:~$ {{command}}
